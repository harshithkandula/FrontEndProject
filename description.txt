

PROJECT
Problem Statement:


Anna runs an Architectural firm. To manage her client meetings and projects, she needs project management software which will be developed by Henna Solutions, a software company in her town. The company has created web pages for the project to interact with the interfaces for various inputs and outputs. For the data entered by the User in the web pages, the company needs to implement validation for the data.


Tasks:


Perform the below activities as a solution:


In Client, Creation Web Page validate correctness for name, email.
In Client Meeting, Schedule Web Page, validate the correctness of the date for booking an appointment
